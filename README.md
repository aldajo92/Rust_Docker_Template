# Rust Project Template

## Project Structure:
```
├── app                     <- Rust Cargo Project
│   ├── Cargo.lock
│   ├── Cargo.toml
│   └── src                 <- Rust sources
│       └── main.rs
├── Dockerfile              
```

## Usage via scripts
```Bash
./scripts/build.sh

./scripts/bash.sh

./scripts/run.sh
```
